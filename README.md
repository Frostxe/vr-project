# VR Darts Game

## Project Description

This project is an early alpha VR darts game built using Godot and OpenXR.
The player can throw darts at a target board using VR controllers and interact with the environment.

## Idea and Goal

The goal of this project is to create an immersive VR darts experience where users can practice aiming and throwing mechanics in a virtual environment.

## Steps Completed

* Setup Godot project with OpenXR
* Created VR player with camera and controllers
* Built simple darts environment
* Added dartboard and dart objects
* Implemented basic throwing mechanics
* Added collision detection with the target
